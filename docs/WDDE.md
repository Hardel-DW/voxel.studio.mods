# Mods - VDDE
Ce mods est est une dépendences direct de Studio, il permet d'étendres certains concepts de Data Driven de Minecraft.

## Fonctionnalités
- **Tags**: La clefs "exclude" (List Identifier ou TagEntry) permet d'exclure des éléments d'un tags, de plus la clefs values n'est plus obligatoire. L'exclusion s'applique aprés la fusion de toutes les valeurs.
